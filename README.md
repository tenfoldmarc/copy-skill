# Copy Skill for Claude Code

Write high-converting copy for sales pages, landing pages, upsell pages, VSLs, webinars, cold emails, Meta ads, challenges, lead magnets, and more.

Trained on 100+ legendary sales letters from Gary Halbert, Eugene Schwartz, Dan Kennedy, Clayton Makepeace, John Carlton, Gary Bencivenga, and other copywriting legends.

## Install

**Easiest way:** Open Claude Code and say:

> Install this skill for me: https://github.com/tenfoldmarc/copy-skill

Claude will handle the rest.

**Or do it yourself:**

Mac/Linux:
```bash
git clone https://github.com/tenfoldmarc/copy-skill.git ~/.claude/commands/copy
```

Windows:
```bash
git clone https://github.com/tenfoldmarc/copy-skill.git %USERPROFILE%\.claude\commands\copy
```

## How It Works

1. Type `/copy` in Claude Code
2. Tell it who the copy is for and what you're writing
3. Claude loads the frameworks and any saved partner profiles
4. It writes copy using proven structures — not generic AI output

## Partner Profiles

The skill gets smarter per client. The first time you use it, it interviews you about your business and creates a partner profile. For agencies with multiple clients, you build up a library of profiles over time.

Partner profiles are saved in the `partners/` folder. Use `partners/_template.md` to create them manually, or let Claude interview you.
